# Grab Item Game

Grab Item Game теперь готова как браузерная версия: сайт с игрой, запускаемой прямо в браузере.

---

## Что добавлено

- `index.html` — веб-страница для игры
- `style.css` — стили оформления
- `game.js` — логика игры на JavaScript
- `.gitignore` — игнорирование временных файлов

Оригинальный файл `GrabItemGame.py` остаётся в проекте как Python-версия.

---

## Как запустить сайт локально

Откройте файл `index.html` в браузере. Игра должна сразу отобразиться и работать.

Если хотите использовать локальный сервер, выполните из папки проекта:

```bash
python -m http.server 8000
```

Затем откройте в браузере:

```text
http://localhost:8000
```

## Как опубликовать на GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите в него все файлы из этой папки (`index.html`, `style.css`, `game.js`, `GrabItemGame.py`, `README.md`, `.gitignore`).
3. В GitHub откройте `Settings` → `Pages`.
4. В разделе "Build and deployment" выберите ветку `main` или `master` и папку `/ (root)`.
5. Сохраните.
6. Через несколько минут сайт будет доступен по адресу типа:

```text
https://<ваше_имя>.github.io/<имя_репозитория>/
```

---

## Как загрузить проект в GitHub (если установлен Git)

Если у вас установлен Git, выполните:

```bash
git init
git add .
git commit -m "Добавить браузерную версию Grab Item Game"
git branch -M main
git remote add origin https://github.com/<ваше_имя>/<имя_репозитория>.git
git push -u origin main
```

Если Git не установлен, можно просто загрузить файлы через веб-интерфейс GitHub.

## Игровое управление

- `W` / `↑` — вверх
- `S` / `↓` — вниз
- `A` / `←` — влево
- `D` / `→` — вправо
- `SPACE` — старт
- `R` — рестарт на экране Game Over
- `ESC` — выйти в меню

---

## Примечание

Веб-версия работает как стандартный сайт и готова для публикации на GitHub Pages. Оригинальный Python-файл можно оставить как демонстрацию версии для локального запуска.
